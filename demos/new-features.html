HIGH IMPACT 1. Offline Detection

Automatically detect when user goes offline/online

_.onOffline(() => _.notify('No internet!', 'error'));
_.onOnline(() => _.notify('Back online!', 'success'));
console.log('Online:', _.isOnline());
Status: Checking...

HIGH IMPACT 2. Clipboard Read

Read clipboard contents (requires user permission)

const text = await _.paste();
console.log('Clipboard:', text);

HIGH IMPACT 3. WebSocket Wrapper

Simple WebSocket wrapper with auto-reconnect

const socket = _.socket('wss://echo.websocket.org');
socket.on('message', (data) => console.log(data));
socket.send('Hello WebSocket!');

Note: WebSocket requires a live server. Example shown above.

HIGH IMPACT 4. Promise Chain UI (Loading State)

Automatically show loading states for async operations

_('#saveBtn').loadingState(promise, {
    loading: 'Saving...',
    success: 'Saved!',
    error: 'Failed!'
});

HIGH IMPACT 5. Share API

Native share functionality (mobile + modern browsers)

_.share({
    title: 'YakaJS',
    text: 'Better than jQuery!',
    url: window.location.href
});

PERFORMANCE 6. Batch DOM Updates

Prevent layout thrashing with batched updates

    _('#a').css('color', 'red');
    _('#b').css('color', 'blue');
    // All applied in one reflow
});

    '/fonts/custom.woff2'
