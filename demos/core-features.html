Explore YakaJS's powerful DOM manipulation capabilities with interactive examples. Each example shows the code and provides an interactive demo you can try.

1. Selector Methods

Select elements using .get(), .first(), .last(), .eq()

Code:

// Select elements
_('.item').get(0)     // First element (DOM)
_('.item').first()    // First element (YakaJS)
_('.item').last()     // Last element (YakaJS)
_('.item').eq(1)      // Element at index 1

2. Content Manipulation

Modify content with .text(), .html(), .val()

Code:

// Get/Set text
_('#output').text('Hello World');

// Get/Set HTML
_('#output').html('<b>Bold</b>');

// Get/Set input value
_('#myInput').val('New value');

3. Class Manipulation

Control element classes with .addClass(), .removeClass(), .toggleClass()

Code:

// Add class
_('#box').addClass('active');

// Remove class
_('#box').removeClass('active');

// Toggle class
_('#box').toggleClass('active');

4. Attributes & Data

Manage attributes with .attr() and data attributes with .data()

Code:

// Set attribute
_('#link').attr('href', 'https://...');

// Get attribute
const href = _('#link').attr('href');

// Set data attribute
_('#box').data('count', 5);

// Get data attribute
const count = _('#box').data('count');

5. Event Handling

Attach events with .on(), .click(), .hover()

Code:

// Click event
_('#btn').on('click', () => {
    alert('Clicked!');
});

// Multiple events
_('#btn').click(() => {
    console.log('Click handler');
});

// Hover events
_('#box').hover(
    () => console.log('Enter'),
    () => console.log('Leave')
);

6. Show/Hide Elements

Control element visibility with .show(), .hide(), .toggle()

Code:

// Show element
_('#element').show();

// Hide element
_('#element').hide();

// Toggle visibility
_('#element').toggle();

// Check if visible
if (_('#element').is(':visible')) {
    console.log('Visible!');
}

7. Method Chaining

Chain multiple operations together for clean, efficient code

Code:

// Chain multiple operations
_('#element')
    .addClass('active')
    .text('Updated!')
    .attr('data-state', 'changed')
    .fadeIn(500);
