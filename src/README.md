## About

This is the unminified, human-readable source code with all comments and formatting intact. Use this for:

- Development and debugging
- Understanding the library internals
- Contributing to the project
- Creating custom builds

## Development

```bash
# Make changes to src/yaka.js
# Then build the distribution files:

npm run build        # Standard build
npm run build:ultra  # Ultra-compressed
npm run build:hyper  # Maximum compression
```
